What data structure used to perform recursion?

Stack is suggested, due to its Last-in First-Out properties. Recursion is make used of stack to store the return addresses of the functions calls.

Every recursive function have its equivalent iterative function.

Pros and Cons of Recursive methods

New storage location for variables are allocated on the stack when a recursive call is made. As each call returns, the old variables and parameters are removed from the stack (pop out from the stack).
Thus, recursion usually consumed more memory and are generally slow.

But the good thing is, recursive solution is much more simpler and takes less time to write and maintain.
