Comparator Interface in Java

In Java, a Comparable class can implements the java.util.Comparator interface.

Comparable and Comparator

If the objects of a class have natural order. For instance, String, Date, etc. We can let that class implement the Comparable interface.

The Comparator interface is much more flexible, since it enables you to define a new class which contains the compare(Object, Object) method to compare two objects of other classes. The comparator interface has two methods, which are compare and equals.
