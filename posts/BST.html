Binary Search Tree

Unlike the linear data structures like arrays, stacks, queues and linked lists, a binary tree is a hierarchical structure. It is either empty or consists of an element, it's called the root, and two distinct binary trees, called the left subtree and right subtree.

Implementation of Tree in Java

Here is how we decleared a class called TreeNode,

class TreeNode<E> {
    E element;
    TreeNode<E> left;
    TreeNode<E> right;
    public TreeNode(E o){
        element = o;
    }
}

How to search for an element in Binary Tree?

To search for a specified element, we start from the root and scan down from it until a match is found, or until it arrive at an empty subtree. The following steps are:
  • If that element is less than current element, assign the left subtree to current.
  • If that element is greater than current element, assign the right subtree to current.
  • If that element is equal to current element, return true.
public boolean search(<E> element) {
    //We start from root of the tree
    TreeNode<E> current = root;
    while (current != null)
        if (element < current.element) {
            current = current.left; // go left if smaller
        }
        else if (element > current.element) {
            // go right if greater
            current = current.right;
        }
        else
            return true;
    }
}

How to insert an element into Binary Tree?

We can locate the parent for the new node if a binary tree is empty, we create a root node with the new element. Otherwise, we locate the parent node for the new element node.

If the new element is less than the parent element, the node for the new element will become the left child of the parent.

If the new element is greater than the parent element, the node for the new element will become the right child of the parent.
