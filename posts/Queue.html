What's Queue?

A queue is a first in first out (FIFO) data structure. The item that goes in first will be the first one to come out.

The features of Queue

In Java Collections Framework, the Queue interface allows us to do these operations:
  • Offer(element: E): Inserts an element to the back of queue
  • Poll(): Dequeue the element at the head of this queue
  • Remove(): Retrieves and removes the head of this queue, throws exception if this queue is empty.
  • Peek(): Get the value of the head element without remove it.
  • Element(): Searching for element in this queue, without dequeue the head element

Priority Queue

A priority queue is a special type of queue in which each element is arranged according to its priority. In case the elements with the same priority occurred, they will served according to their order in the queue.

What is the difference between priority queue and a queue?

In a queue, the FIFO rule is implemented. In a priority queue, the values are dequeue based on their priority. The element with the highest priority will be remove first.

The Features of Priority Queue

  • PriorityQueue(): create queue class.
  • PriorityQueue(initialCapacity : int) : create a priority queue with specified initial capacity
  • PriorityQueue(c: Collection) : create a priority queue with specified collection.
